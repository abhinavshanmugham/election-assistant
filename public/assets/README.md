# Party assets (flags and symbols)

Place image files here so the app can show party flags and symbols.

**Flags** (`flags/`): dmk.png, aiadmk.png, tvk.png, ntk.png, pmk.png, dmdk.png, mdmk.png, vck.png, bjp.png, inc.png

**Symbols** (`symbols/`): rising_sun.png, two_leaves.png, whistle.png, farmer_plough.png, mango.png, drum.png, matchbox.png, pot.png, lotus.png, hand.png

Files are referenced from `src/data/parties.js` and served from the app root (e.g. `/assets/flags/dmk.png`).
